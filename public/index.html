Rejseplanen | Google Calendar

Ralender is an online event planning tool that combines functionalities from Rejseplanen and Google Calendar into one, where events created through Ralendar automatically adds travel time from Rejseplanen, which is then inserted into the users Google Calendar.

Ralendar only requires authentication from Google Calendar to be able to insert events into Google Calendar. Ralendar does not read users' calendars, but still needs to be authenticated in order for Ralendar to work.
